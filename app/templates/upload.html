Data Requirements

Your CSV file should include the following columns:

  • Type - Equipment or component type
  • Air temperature [K] - Operating environment temperature
  • Process temperature [K] - Process temperature
  • Rotational speed [rpm] - Machine rotational speed
  • Torque [Nm] - Applied torque
  • Tool wear [min] - Tool wear time in minutes

If your data has different column names, the system will attempt to use the best matching features.
